News Sentiment Classification using NLP & Web Scraping

Developing a sentiment analysis system using NLP and Machine Learning to classify text sentiment as positive, negative, or neutral.

This project focuses on classifying news articles based on sentiment (Neutral, Opposing, or Supporting) using Natural Language Processing (NLP). News content is scraped from online sources using BeautifulSoup, then preprocessed with NLTK and Sastrawi for tokenization, stopword removal, and stemming. The classification is performed by matching keywords against predefined sentiment-based word lists.

The system provides insights into how different media outlets present information and can be applied in media bias analysis, public opinion research, and automated news categorization.
