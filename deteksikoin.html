Coin Detection with Edge Detection

Developing a coin detection system using OpenCV & Image Processing.

🛠 Technologies: Python 🐍, OpenCV 👁, NumPy 🔢, Matplotlib 📊, Tkinter 🎨

🎯 Skills: ✅ Computer Vision, ✅ Image Processing, ✅ Object Detection

This project develops an automated coin detection system using OpenCV and image processing techniques. Users can select an image containing coins, and the system will analyze and detect them based on shape and size.

The system performs grayscale conversion, Gaussian blurring, adaptive thresholding, and morphological transformations to enhance detection accuracy. It then identifies circular objects (coins) using contour detection and visualizes the results.

Additionally, a guide feature displays correct and incorrect coin placement references to help users ensure proper positioning before detection. This project is useful for coin counting machines, currency validation systems, and educational purposes in image processing
