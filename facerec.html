This project aims to develop an automated face detection system capable of operating on both static images and real-time video. The Haar Cascade algorithm is used to quickly and efficiently detect facial features under various lighting conditions. Additionally, Deep Learning technology using a CNN model is implemented to enhance face detection accuracy in more complex scenarios.

This system is expected to be applicable in fields such as security, surveillance, and facial-based interactions across various technological domains.
