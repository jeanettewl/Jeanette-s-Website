Ticketing Website for Mini Dance Show 2024

This web-based application was developed for the Mini Dance Show 2024, a student-organized dance event. The main purpose of the site is to provide a smooth and efficient way to handle ticket bookings and event entry validation.

🛠 Technologies Used: PHP Native 🐘, HTML5, CSS3 🎨, JavaScript ⚙, PHPMailer ✉, QRCode.js 🔲

🎯 Key Skills: ✅ Authentication System, ✅ Ticket Management, ✅ Email Integration, ✅ QR Code Generation

System Overview

The system is divided into two main user roles: Customer and Administrator.

Customer Features:

  • Ticket booking form with customer data input
  • Automatic payment instructions display
  • Email confirmation after successful order
  • Digital ticket generation with a unique QR code
  • Customers can download and save their e-ticket for event entry

Admin Features:

  • Secure login and authentication
  • Dashboard for managing customer data and ticket sales
  • Manual or automated payment status updates
  • Email ticket dispatch using PHPMailer
  • QR Code validation for ticket check-in on the event day

The website was developed without any frameworks, using pure/native PHP. It features organized folders for maintainability, including separate directories for admin, pelanggan (customers), uploads, and includes.

Ideal for event organizers looking for a simple yet effective ticketing solution, this project demonstrates core backend handling, frontend structuring, and integration with real-time technologies like QR scanning and mailing systems.
